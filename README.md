# samp-cli

CLI samp client (suckless-like), that can be launch samp and arizona. 

Install:
```
sudo make install clean
```

Usage:
```
#Launch samp
samp-cli

#Launch arizona
samp-cli -a

#Display saved servers
samp-cli -l

#Set priority nickname. This nickname will be used instead that in config
samp-cli -n
```
